Frequently Asked Questions

Can I add page numbers to password-protected PDFs?

You'll need to remove password protection first using our PDF protection tool, then you can add page numbers normally.

Can I skip certain pages when numbering?

Yes, you can specify page ranges to skip, such as title pages, covers, or table of contents pages.

Will adding page numbers affect my PDF quality?

No, our tool preserves the original PDF quality while adding page numbers as a new layer.

Can I use different numbering formats in the same document?

Yes, you can use different formats for different sections, like Roman numerals for front matter and Arabic numbers for main content.

Is there a limit to file size for page numbering?

Our tool supports files up to 50MB. For larger files, consider compressing first or processing in sections.
